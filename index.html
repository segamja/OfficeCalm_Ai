    Mindly를 홈 화면에 설치하고 앱처럼 사용해 보세요.

    홈 화면에 설치하기

    Chrome에서는 아래 홈 화면에 설치 버튼으로 브라우저 설치 창이 열립니다. iPhone은 Safari에서 수동으로 추가해 주세요.

    설치 준비 중

    브라우저가 설치 가능 상태를 확인하는 중입니다. 잠시 후 다시 홈 화면에 설치 버튼을 눌러 주세요. (Chrome · HTTPS · 첫 방문 후 1~2회 새로고침 필요할 수 있음)

    iPhone (Safari)

    1. Safari에서 이 페이지를 엽니다 (카카오톡 등 앱 내 브라우저는 안 됩니다)
    2. 하단 공유 버튼(□↑)을 누릅니다
    3. 홈 화면에 추가를 선택합니다
    4. 이름 Mindly 확인 후 추가를 누릅니다

    Android (Chrome) — 자동 설치

    상단 배너 또는 설정의 홈 화면에 설치 버튼을 누르면 Chrome 설치 프롬프트가 표시됩니다.

    앱 내 브라우저 감지

    카카오톡·인스타그램 등 앱 안에서는 설치가 되지 않습니다. Chrome 또는 Safari에서 아래 주소를 직접 열어 주세요.
