# Mindly UX/UI Improvement Specification v1.0

> Project: **Mindly** (Formerly OfficeCalm AI)

## 목적

현재 MVP는 기능적으로 완성도가 높지만, 사용 경험은 '대시보드'에 가깝다.
이번 개선의 목표는 기능 추가가 아니라 **AI Mental Coach** 경험을 만드는
것이다.

------------------------------------------------------------------------

# UX 철학

기존 흐름

사용자 → 기능 선택 → 결과 확인

목표 흐름

사용자 → Mindly가 먼저 인사 → 상태 파악 → AI 추천 → 회복 활동 → 칭찬과
성장

Mindly는 명상 앱이 아니라 **직장인을 위한 AI Recovery Coach**로 느껴져야
한다.

------------------------------------------------------------------------

# 우선순위

1.  홈 화면 개선 (★★★★★)
2.  AI 경험 강화 (★★★★★)
3.  감성 UX (★★★★☆)
4.  XP/레벨 시스템 (★★★★☆)
5.  향후 AI 연동 고려 (★★★☆☆)

------------------------------------------------------------------------

# 2. 최초 실행(Onboarding)

## 최초 실행 확인

앱 실행 시 localStorage.nickname 존재 여부를 확인한다.

### nickname이 없는 경우

온보딩 화면 표시

    안녕하세요 😊

    저는 Mindly입니다.

    앞으로 함께 마음을 돌보는 여정을 시작해볼까요?

    당신을 어떻게 불러드리면 될까요?

    [ 닉네임 입력 ]

    [ 시작하기 ]

### 저장

    localStorage.nickname

### 이후

모든 화면에서

    {nickname}님

형태로 표시한다.

예)

-   좋은 아침입니다. 홍민님 😊

### 설정

설정 화면에서 닉네임 변경 가능

### 향후 확장

Firebase 로그인 시

localStorage → Firebase displayName으로 쉽게 교체 가능하도록 모듈화한다.

# 3. 홈 화면 개선

기존

-   XP
-   Mind Energy
-   Streak

홈 최상단에 Hero Card 추가

    🌤 좋은 아침입니다.

    홍민님

    오늘 컨디션
    76%

    어제보다 +8%

    오늘의 미션

    □ AI 코칭
    □ 호흡
    □ 감사일기

    [ 오늘 시작하기 ]

요구사항

-   Hero Card가 첫 화면
-   시간대별 인사말 변경
-   Start 버튼 → AI 화면 이동

# 4. 오늘의 AI 메시지

Hero Card 아래 랜덤 메시지 표시

예시

-   오늘도 충분히 잘하고 있습니다.
-   회복은 작은 휴식에서 시작됩니다.
-   잠깐 쉬어도 괜찮습니다.
-   오늘도 함께 성장해봐요.

# 성공 기준

사용자는 앱을 사용한 후 다음과 같이 느껴야 한다.

-   Mindly가 나를 이해한다.
-   Mindly가 먼저 챙겨준다.
-   Mindly가 내 성장을 응원한다.
-   Mindly와 매일 대화하고 싶다.

최종 목표는 명상 앱이 아니라 **직장인을 위한 AI Recovery Coach**를
만드는 것이다.
