# OfficeCalm AI

만성 번아웃과 직무 스트레스에 시달리는 직장인을 위한 AI 기반 맞춤형 명상 및 즉각적 멘탈 케어 웹앱 MVP입니다.

백엔드 없이 브라우저만으로 동작하며, 사용자 데이터·XP·감사일기·레벨은 **localStorage**에 저장됩니다.

**라이브 데모:** [https://segamja.github.io/OfficeCalm_Ai/](https://segamja.github.io/OfficeCalm_Ai/)

**MVP 버전:** v1.3 (2026-07-11)

## 기술 스택

| 영역 | 선택 | 이유 |
|---|---|---|
| 마크업 | HTML5 | 별도 설치 없이 브라우저 즉시 실행 |
| 스타일 | CSS3 (Vanilla) | 빌드 도구 없이 순수 CSS로 유지보수 단순화 |
| 로직 | JavaScript ES6 (`<script>` 태그) | `import/export` 없이 로컬/HTML 더블클릭 실행 가능 |
| 데이터 저장 | **localStorage** | 서버·DB 없이 기기 내 영속화, 프라이버시 보호 |
| 오디오 | HTML5 `<audio>` | 네이티브 MP3 재생 |
| 알림 | Web Notification API | 브라우저 권한 기반 로컬 리마인더 |
| 배포 | GitHub Pages | 무료 정적 호스팅 |

### SQLite를 쓰지 않는 이유

브라우저 단독 실행(Serverless) MVP에서는 **SQLite 파일을 직접 쓸 수 없습니다.**  
SQLite 연동은 `sql.js` + WASM 등 추가 라이브러리가 필요하고, GitHub Pages 정적 배포와 1인 개발 공수 측면에서 **localStorage JSON 구조**가 가장 적합합니다.

향후 앱 확장(네이티브 앱, Electron, 백엔드 연동) 시 SQLite/PostgreSQL로 마이그레이션할 수 있도록 데이터 스키마를 문서화해 두었습니다.

## 레이아웃 (5탭 단일 화면)

**PC·모바일 공통**으로 하단 5탭 바를 사용하며, 활성 탭 패널 1개만 전체 화면에 표시됩니다. 앱은 최대 640px 너비로 중앙 정렬되어 모바일 앱과 동일한 UX를 제공합니다.

| 화면 | 동작 |
|------|------|
| **모든 화면** | 하단 고정 5탭 — 활성 탭 1개만 전체 화면 표시 |

### 5개 탭 구성

| 탭 | 포함 콘텐츠 |
|----|-------------|
| **홈** | 로고, 서브타이틀(직장인을 위한 AI 멘탈 케어), XP·레벨·Mind Energy, 스트릭, 알림, 오늘의 명상 완료, **음악 재생 ON/OFF** + 플레이어 |
| **AI 진단실** | 맞춤 코칭 스크립트, 타이핑 출력, 완료 후 **브리드로 이동** CTA |
| **오피스 브리드** | 호흡 버블 + 힐링 이미지 갤러리 (8초 랜덤 전환) |
| **라이브러리** | 배경음·효과음 재생 + 라이브러리 패널 안내 (AI 출력창 미사용) |
| **감사일기** | 감사일기 작성 + 히스토리 |

- 실행 시 기본 탭: **홈**
- 마지막 선택 탭은 `sessionStorage` (`officeCalm_active_tab`)에 저장
- 음악 재생은 **홈 탭에서 ON**으로 켠 뒤, 라이브러리·AI에서 트랙 재생 가능
- 음악 OFF 시 플레이어 UI 숨김 (`hidden` 속성 + CSS 이중 보장)

## 주요 기능

### 1. 홈 (메인 대시보드)
- XP·레벨·Mind Energy·스트릭 한눈에 확인
- 오늘의 명상 완료, 알림 리마인더
- **음악 재생 ON/OFF** 토글 및 미니 플레이어 (볼륨·일시정지·정지)

### 2. AI 스트레스 진단실
- 상황별 **맞춤 명상 코칭 스크립트** + 타이핑 효과
- 스크립트 타이핑 완료 후 **브리드로 이동** CTA 표시
- **음악 재생 ON** 상태일 때만 명상 음악 자동 재생
- 퀵 프리셋 4종 (회의 5분 전, 상사 잔소리, 출근길 무기력, 야근 번아웃) + 자유 입력
- **모든 AI 기능 무료** — 결제/체험 팝업 없음
- AI 프리셋(예: 회의 5분 전)은 **코칭 스크립트** — 라이브러리의 배경음과 역할이 다름

### 3. 오피스 브리드
- 4초 들이쉬기 · 2초 멈추기 · 4초 내쉬기
- 시각적 호흡 가이드 버블
- 힐링 이미지 갤러리 (8초 랜덤 전환)

### 4. 감사일기
- 매일 감사한 일 작성·수정
- **히스토리**에서 과거 기록 열람
- 하루 1회 작성 시 **+20 XP**

### 5. XP · 레벨 · Mind Energy
- 활동별 경험치 적립 → 레벨업
- **Mind Energy Score**: 스트릭·명상·감사일기·레벨을 종합한 회복 지표 (0~100%)
- 레벨업 시 **상금형 축하 모달** 표시

### 6. 콘텐츠 라이브러리
- **배경음·효과음** 재생 전용 — AI 출력창에는 쓰지 않음
- 재생 시 라이브러리 패널에 짧은 세션 안내 표시 (예: "사무실 백색소음 재생 중 — 편안히 호흡하세요")
- 호흡 가이드는 **브리드 탭**에서 이어가도록 UI 안내
- 오디오 목록 + 레벨 게이트 (플레이어 UI는 홈 탭)

| 트랙 | 시간 | 해금 조건 |
|------|------|-----------|
| 사무실 백색소음 | 5분 | 기본 (Lv.1) |
| 책상 앞 스트레칭 | 3분 | 기본 (Lv.1) |
| 야근 후 딥슬립 | 20분 | **Lv.3** (마음 수호자) |
| 번아웃 회복 명상 | 15분 | **Lv.5** (칼름 리더) |

### 7. 데일리 리추얼 & 레벨 게이트
- 연속 명상 스트릭
- 레벨 미달 콘텐츠 클릭 시 **레벨 안내 모달** (XP 획득 방법 안내)
- 레벨업 시 잠금 UI 자동 해제

## XP 보상 체계

| 활동 | XP | 비고 |
|------|-----|------|
| AI 명상 스크립트 생성 | +15 | 음악 ON 시 자동 재생 포함 |
| 오늘의 명상 완료 | +25 | 하루 1회 |
| 감사일기 작성 | +20 | 하루 1회 (최초 저장) |
| 오디오 재생 | +10 | 홈에서 음악 ON 후 트랙 재생 시 |

### 레벨 구간

| 레벨 | 필요 XP | 칭호 |
|------|---------|------|
| Lv.1 | 0 | 새싹 직장인 |
| Lv.2 | 50 | 회복 루키 |
| Lv.3 | 120 | 마음 수호자 |
| Lv.4 | 220 | 밸런스 마스터 |
| Lv.5 | 350 | 칼름 리더 |
| Lv.6 | 520 | 멘탈 멘토 |
| Lv.7 | 750 | 오피스 힐러 |
| Lv.8 | 1050 | 평온의 달인 |

### Mind Energy Score 산출

단순 활동량이 아니라 **정신적 회복 수준**을 나타내는 지표입니다.

```
기본 35
+ 스트릭 × 4 (최대 24)
+ 오늘 명상 완료 시 +18
+ 오늘 감사일기 작성 시 +15
+ 레벨 × 3 (최대 18)
+ XP 누적 보너스 (최대 10)
→ 0~100%로 정규화
```

## AI vs 라이브러리 역할

| | AI 스트레스 진단실 | 콘텐츠 라이브러리 |
|--|--|--|
| 목적 | 맞춤 코칭 스크립트 + 브리드 유도 | 배경음·효과음 재생 |
| 예시 | 「회의 5분 전」→ 긴 코칭 텍스트 | 「사무실 백색소음」→ 5분 집중 사운드 |
| 출력 | AI 출력창 (타이핑) | 라이브러리 패널 안내 문구 |

## AI 명상 + 자동 음악 매핑

| 상황 | 자동 재생 트랙 |
|------|----------------|
| 회의 5분 전 | 사무실 백색소음 |
| 출근길 무기력 | 책상 앞 스트레칭 |
| 상사 잔소리 / 야근 번아웃 / 자유 입력 | 번아웃 회복 명상 |

레벨이 부족하면 대체 트랙으로 재생됩니다 (번아웃 회복 → 책상 앞 스트레칭, 딥슬립 → 백색소음).  
**홈 탭에서 음악 재생을 ON**으로 설정해야 실제 재생됩니다.
